### setup
- open xcode 
- window -> organizer
- distribute app -> app store connect

or 

- run workflow and generate .ipa
- upload .ipa in transporter
- wait for build to upload and install

### presetup
run
```bash
expo prebuild --clean
```

or 

```bash
eas build -p ios --profile production
```
